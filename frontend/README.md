# RestaurantConnect Frontend

This project is a React frontend application that connects to the existing Spring Boot backend for managing restaurant reservations.

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)
- [API Endpoints](#api-endpoints)
- [Contributing](#contributing)
- [License](#license)

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd restaurantconnect-frontend
   ```

3. Install the dependencies:
   ```
   npm install
   ```

## Usage

To start the development server, run:
```
npm run dev
```

Open your browser and go to `http://localhost:3000` to view the application.

## API Endpoints

The frontend connects to the following API endpoint:

- `GET /api/reservations`: Fetches the list of reservations.
